# calender-with-memo
下にメモ帳がついた書き込み式カレンダーです。  
カレンダーとメモ欄の大きさは自由に調整できます。  
メモ欄はトグルですことも可能です。  
  
Simply writeable calendar with a notepad  
Calendar and memo area sizes are adjustable.  
The memo area can also be toggled.
